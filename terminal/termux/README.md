# Termux
This is termux configuration.

- install necessary dependencies
- use zsh as default shell, setup zsh environment
- custom termux.properties

## install

    bash -c "$(curl -fsSL https://gtk.pw/termux)"
    # or
    bash -c "$(wget https://gtk.pw/termux -O-)"
